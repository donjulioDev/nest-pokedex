## Description
1. Clonar el repositorio
2. Ejecutar el comando
```bash  
yarn install
```
3. Tener Nest CLI instalado
```bash  
pnpm i -g @nestjs/cli 
```
4. Levantar la base de datos
```bash  
docker-compose up -d
```
